MYSQL DB
=========

This role installs MySQL database on a CentOS 8 system.


Requirements
------------

None.


Dependencies
------------

Python must be installed.


Example Playbook
----------------

Example of implementation:

    - hosts: $SERVERS
      roles:
         - mysql_db
